# Travelogue Theme Design Reference

## Key Design Patterns

### 1. Full-Screen Hero
- Edge-to-edge background image/video
- Centered text with large serif typography
- Minimal navigation (just menu icon and search)
- Scroll indicator at bottom

### 2. Grid Layout (Below Hero)
- 4-column grid of full-bleed image tiles
- Each tile is a clickable card with:
  - Full background image covering entire tile
  - Text overlay in top-left corner
  - Bold keyword + regular text pattern (e.g., "GO **EXPLORE**")
  - Subtle hover effect (zoom or darken)
- No gaps between tiles (edge-to-edge)
- Various tile sizes (some span 2 columns)

### 3. Typography Pattern
- First word(s) in light weight
- Key word in BOLD/heavy weight
- All uppercase
- White text over images
- Subtle text shadow for readability

### 4. Color Scheme
- Full-color photography as backgrounds
- White text overlays
- No cards or containers - content floats directly on images
